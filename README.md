# django-mysite

Django Tutorial mysite in Django LTS

Django 公式チュートリアルのプロジェクト mysite をLTS版Djangoで作成した時の生成ファイルを比較のために保存。
LTSバージョンアップ時のベースとして利用。


## pip

packages : managed by pip

* $ pip install -r requirements.txt
